You have been tasked with completing the following user story.

Display a list of customers with details

Description

A user should be able to click on the 'Customer Directory' link where they will be routed to /customers. On the customers route the user should see a list of customers returned using a GET request.

Acceptance Criteria

  • Users can navigate to /customer using the Customer Directory link located on the homepage
  • Users should see a list of customers with their details hidden by default
  • Users should be able to click on a customer to expand the customer details
  • Follows the provided design as closely as possible considering you can't inspect the design
